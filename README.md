# Network

A simple Android utility app that acts as a shortcut to the system's hidden **Radio Info** settings screen (`*#*#4636#*#*`).

Some Android devices and custom dialer apps block the classic dialer codes used to access advanced cell configuration menus. This app bypasses those limitations, taking you straight to the configuration page.

## Features

- **Instant Launch**: Opens the settings menu immediately upon opening the app.
- **Privacy Friendly**: Requests zero device permissions. 
- **Modern Icon**: Includes an adaptive launcher icon for modern Android devices.

## Requirements

- Android 7.0 (Nougat) or higher.
